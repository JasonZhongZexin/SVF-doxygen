\doxysubsection{Detailed Description}
Represents the \mbox{\hyperlink{class_s_v_f_1_1_p_a_g}{PAG}} of a function loaded externally (i.\+e. from file). It\textquotesingle{}s purpose is to be attached to the main \mbox{\hyperlink{class_s_v_f_1_1_p_a_g}{PAG}} (almost) seamlessly.
