 \mbox{\hyperlink{namespace_s_v_f}{SVF}} is a source code analysis tool that enables interprocedural dependence analysis for LLVM-\/based languages. \mbox{\hyperlink{namespace_s_v_f}{SVF}} is able to perform pointer alias analysis, memory SSA form construction, value-\/flow tracking for program variables and memory error checking. ~\newline
