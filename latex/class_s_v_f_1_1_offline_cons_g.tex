\doxysubsection{Detailed Description}
Offline constraint graph for \mbox{\hyperlink{class_s_v_f_1_1_andersen}{Andersen}}\textquotesingle{}s analysis. In OCG, a \textquotesingle{}ref\textquotesingle{} node is used to represent the point-\/to set of a constraint node. \textquotesingle{}Nor\textquotesingle{} means a constraint node of its corresponding ref node.
