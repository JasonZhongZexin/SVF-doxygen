\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Key, typename Datum, typename Data$>$\newline
class SVF\+::\+DFPTData$<$ Key, Datum, Data $>$}

Data-\/flow points-\/to data structure for flow-\/sensitive analysis as defined by Hardekopf and Lin (CGO 11). Points-\/to information is maintained at each program point (statement). For address-\/taken variables, every program point has two sets\+: IN and OUT points-\/to sets. For top-\/level variables, points-\/to sets are maintained flow-\/insensitively via get\+Pts(var).
