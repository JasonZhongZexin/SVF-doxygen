Detailed Description

template<typename Key, typename Datum, typename Data, typename VersionedKey>
class SVF::MutableVersionedPTData< Key, Datum, Data, VersionedKey >

VersionedPTData implemented with mutable points-to set (Data). Implemented as a wrapper around two MutablePTDatas: one for Keys, one for VersionedKeys.
