\doxysubsection{Detailed Description}
Allocates node IDs for objects and values, upon request, according to some strategy which can be user-\/defined. It is the job of \mbox{\hyperlink{class_s_v_f_1_1_symbol_table_info}{Symbol\+Table\+Info}} to tell the \mbox{\hyperlink{class_s_v_f_1_1_node_i_d_allocator}{Node\+IDAllocator}} when all symbols have been allocated through end\+Symbol\+Allocation.
